# Real-time_signal-analysis
Real-time audio signal analyzer using Python, NumPy, FFT and Matplotlib

This project captures real-time audio from the microphone and analyzes it using signal processing techniques.

 Features

 Record live audio from microphone
Plot audio waveform (Time Domain)
Perform Fast Fourier Transform (FFT)
Display Frequency Spectrum
Understand real-world signal behavior

# Technologies Used

* Python
* NumPy
* Matplotlib
* SoundDevice



 How It Works

1. Captures audio input using microphone
2. Converts sound into digital signal
3. Plots waveform (amplitude vs time)
4. Applies FFT to convert signal into frequency domain
5. Displays frequency spectrum

# Run the Project

Install dependencies:

```bash
pip install numpy matplotlib sounddevice


Run the program:

```bash
python realtime_signal.py
```

---

 Output

* Time-domain signal (waveform)
* Frequency-domain signal (spectrum)
